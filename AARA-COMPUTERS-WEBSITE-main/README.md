# AARA Computers Website

Static website for AARA Computers with all assets organized under `assets/`.

## Structure
- `index.html`, `about.html`, `courses.html`, `services.html`, `gallery.html`, `contact.html`
- `assets/css/styles.css`
- `assets/js/main.js`
- `assets/images/` for all images
- `assets/videos/company-video.mp4`
- `assets/icons/` for favicon and manifest
